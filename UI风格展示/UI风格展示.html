🌃 赛博朋克风格

以"高科技、低生活、暗网感、未来都市"为核心主题，常用黑底+霓虹发光元素（粉、蓝、青），搭配科技网格、故障效果、HUD面板。赛博朋克UI强调冷冽的科技感与压迫性的信息密度。

典型配色
HUD面板 霓虹发光 深色底板 科技网格 故障效果 信息密度高
角色信息面板
典型的HUD风格布局：左侧菜单导航、中间角色展示、右侧数据面板，粉青霓虹配色

💜 霓虹风格

模拟霓虹灯/荧光灯管的发光视觉效果，核心是"高饱和、高亮度、边缘发光、通透感"。多以细线条、边框、描边形式出现，常用于直播、社交平台的UI组件及按钮设计。

典型配色
外发光/辉光 高饱和色彩 半透明线条 边框描边 通透感
直播界面组件套件
粉紫霓虹色调，包含摄像头框、信息条、状态按钮等完整直播UI组件
直播界面组件（蓝青配色）
青蓝霓虹色调，与粉紫形成互补，包含面板框、按钮、标签等
霓虹风格完整UI组件库
包含图表、按钮、数据面板、地图、进度条等数十种UI组件，展示霓虹风格的全面应用
霓虹发光按钮合集
六种颜色的霓虹发光按钮，展示典型的边缘发光效果和线条感

🌸 二次元科幻风格

以二次元立绘为核心元素，背景多为浅色系渐变，界面通透、精致，按钮多为圆角、磨砂玻璃效果。融合科幻元素，营造未来感与二次元美学的结合。

典型配色
二次元立绘 浅色渐变 磨砂玻璃 圆角按钮 通透精致 科幻元素
游戏启动界面
故障效果（Glitch）+ 紫粉渐变 + 大字体标题，融合科幻与二次元感
角色档案面板
浅紫渐变背景 + 精致立绘 + 属性数据面板，体现二次元科幻的精致感
探访遗迹 · 关卡选择
节点式关卡结构 + 冷灰紫色调 + 水墨风插画缩略图

⚔️ 暗黑写实风格

以暗色调、写实渲染为核心，常见于ARPG、魂类游戏。界面色彩以黑、暗红、金色为主，强调材质质感和厚重感，UI元素往往带有金属、石刻、皮革等纹理。

典型配色
写实渲染 暗色调 金属质感 厚重感 暗红+金 信息简洁
角色面板
暗黑风全屏角色展示，左侧简洁角色信息，右侧装备/社交入口，暗红+金色点缀
任务关卡界面
旧地图纹理 + 金色光束 + 写实道具图标，古典探险氛围
武器详情面板
暗红辉光 + 血滴效果 + 金属质感武器渲染，极致暗黑美学

🏯 古风 / 国风

以传统元素（毛笔字、水墨、祥云、印章）为核心，颜色多为墨黑、米白、朱砂、石青，质感偏宣纸、木纹。国风UI追求东方意境与留白美学。

典型配色
水墨画风 毛笔书法 宣纸质感 留白意境 朱砂石青 东方美学
武器/武学系统多界面合集
9宫格展示古风游戏多个功能界面：活动、武器详情、技能树等，水墨背景+毛笔字标题
角色系统多界面合集
9宫格展示角色界面：数据面板、装备、宝石镶嵌、服装等，淡雅水墨风+仙侠元素

🎀 Q版二次元

以Q版（SD头身比）角色立绘为核心，色彩明亮活泼，界面元素圆润、可爱，按钮多为糖果色渐变，整体视觉轻松愉悦，适配休闲、养成类游戏。

典型配色
Q版角色 糖果色彩 圆润元素 活泼轻松 渐变按钮 可爱风
角色管理面板
粉蓝渐变背景 + Q版角色立绘 + 圆角信息卡片，糖果色系按钮
角色选择 / 皮肤展示
卡片式角色展示 + 涂鸦风标题 + 粉色渐变按钮，视觉活泼生动
冒险任务界面
手账/便签风格布局 + 插画卡片 + 清新淡彩色调，充满日系治愈感
